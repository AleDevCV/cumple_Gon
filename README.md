# 🎂 cumple_Gon

Repositorio para dedicar un feliz cumpleaños a **Gon**. 🎉

## ¿Qué incluye?

- Página web interactiva de dedicatoria de cumpleaños (`index.html`).
- Animación de confeti al cargar la página o al presionar el botón **¡Celebrar!**
- Estrellas parpadeantes de fondo y diseño visual festivo.

## Cómo verlo

Abre el archivo `index.html` directamente en tu navegador, o despliégalo en cualquier servicio de páginas estáticas (GitHub Pages, Netlify, etc.).
